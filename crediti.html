Crediti per Storia di Venezia

Storia di Venezia è un progetto d'un gruppo di studenti americani dal Worcester Polytechnic Institute, a Worcester, Massachusetts, Stati Uniti.
È un progetto del Venice Project Center

Il contenuto del timeline è dall'Atlante storica di Venezia, per Giovani Distefano e pubblicato per Supernova Edizioni srl.

Il timeline usa degli script esterni:
  • l'AngularJS, uno script di Google per creare delle app web interattive. AngularJS è disponibile dalla sua pagina web a AngularJS.org
  • Uno script dal wiki Mozilla Developers' Network. Questo script è disponibile dal wiki a developer.mozilla.org.


Tutti gli altri script sono creati originalmente per questo progetto.
